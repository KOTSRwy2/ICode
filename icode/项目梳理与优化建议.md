# 脑电-核磁可视化平台（PyQt-Fluent-Widgets）重构推进计划与竞赛化方案

## 1. 项目目标与定位（面向比赛/大创）

## 1.1 总体定位
打造一个**轻量化、易上手、低门槛**的脑电（EEG）/核磁（fMRI）可视化平台，面向新手快速完成：

- 数据导入
- 参数配置
- 一键运行
- 多图联动展示
- 数据指标解释
- 结果导出（图片/视频/报告）

核心理念：**“多图 + 数据 + 解释”**，不仅展示图，还要展示可理解的定量信息与用途说明。

## 1.2 比赛导向价值
- **创新性**：脑科学算法结果可视化与解释一体化
- **实用性**：新手友好、流程标准化、可复现实验
- **完成度**：具备完整系统闭环（输入→处理→展示→导出→教学）
- **展示性**：统一专业 UI、全屏交互窗口、可视化看板

---

## 2. 现状基线（基于现有代码）

当前已具备 4 个功能能力（`icode/main.py`）：

1. EEG 源定位可视化（`source_localization.py`）
2. EEG 功能连接可视化（`connectivity_visualization.py`）
3. fMRI 脑区激活定位（`fmri_visualization.py`）
4. fMRI 功能连接分析（`fmri_visualization.py`）

现状特点：
- 计算链路基本可用（MNE / Nilearn）
- 结果形式不统一（3D窗口 + HTML）
- 页面组织较基础，参数说明与解释层不足
- 竞赛所需“专业感、完整系统感”尚需强化

## 3. 推荐重构总架构（PyQt-Fluent-Widgets）

## 3.1 顶层架构
采用 **PyQt5 + PyQt-Fluent-Widgets** 重构为“左侧导航 + 右侧工作区”：

- 左侧：四大独立功能板块（一级导航）
- 右侧：参数区 + 运行区 + 可视化区 + 解释区
- 顶部：任务状态、主题切换、导出入口
- 底部：日志与进度栏

## 4. 四大板块页面设计（统一交互模型）

每个板块统一为：**数据输入 → 参数设置 → 运行 → 全屏交互子窗口 → 多图数据解释**。

## 4.1 板块 1：EEG 源定位
- 输入：`.bdf`
- 关键参数：时长、滤波范围、逆解方法、阈值
- 主图：3D 皮层源定位（可全屏）
- 辅图：时间窗能量曲线、脑区贡献 TopN
- 解释：当前热区对应脑区含义、可能功能解释

## 4.2 板块 2：EEG 功能连接
- 输入：`.bdf`
- 主图：3D 脑网络（可全屏）
- 图增强建议：
  - 线宽 = 连接强度
  - 颜色 = 正/负相关分区
  - 节点大小 = 度中心性
  - 节点标签 = 脑区名称
- 辅图：连接矩阵热力图、强边排名表、网络统计卡片

## 4.3 板块 3：fMRI 激活定位
- 输入：`.nii/.nii.gz`
- 主图：激活图（切片联动 + 阈值滑动）
- 辅图：激活体素数量统计、激活强度分布直方图
- 解释：阈值变化对激活区域范围影响

## 4.4 板块 4：fMRI 功能连接
- 输入：`.nii/.nii.gz`
- 主图：ROI 连接网络图（可全屏）
- 辅图：ROI 相关矩阵热力图、模块划分结果、关键脑区列表
- 解释：ROI 连接关系、潜在网络功能注释

---

## 5. “多图 + 数据 + 解释”指标体系（必须落地）

## 5.1 每个页面最少 4 类内容
1. **主交互图**（3D/网络/切片）
2. **辅助统计图**（热力图/柱状图/趋势图）
3. **定量指标卡**（均值、峰值、TopN、阈值）
4. **解释文本区**（术语解释 + 用途说明 + 注意事项）

## 5.2 统一指标示例
| 模块 | 指标 | 展示方式 | 解释用途 |
|---|---|---|---|
| EEG源定位 | 峰值源强、活跃脑区Top5 | 指标卡 + 柱状图 | 识别关键脑区 |
| EEG连接 | 平均连接强度、强边数量、网络密度 | 指标卡 + 热力图 | 评估脑网络耦合 |
| fMRI激活 | 激活体素数、95分位阈值 | 指标卡 + 分布图 | 量化激活范围 |
| fMRI连接 | ROI数量、强连接比例、模块数 | 指标卡 + 网络图 | 描述功能网络结构 |

## 6. 技术方案建议（可执行）

## 6.1 前端/UI层
- `PyQt5` + `PyQt-Fluent-Widgets`
- Fluent 组件：NavigationInterface、CardWidget、InfoBar、ProgressRing、Dialog
- 统一主题：深色专业风格 + 高对比强调色

## 6.2 算法与可视化层
- EEG：`mne` / `mne-connectivity`
- fMRI：`nilearn` / `nibabel`
- 3D：`pyvistaqt`（EEG 3D）
- 统计图：`matplotlib`（或后续接 `pyqtgraph` 以增强实时性）

## 6.3 工程层
- 异步任务：`QThread` + 任务状态机
- 缓存：中间结果缓存（避免重复重算）
- 导出：
  - 图片：PNG
  - 视频：MP4（关键帧回放）
  - 报告：Markdown/PDF（图 + 参数 + 结论）
